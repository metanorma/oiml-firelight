## Overview

This is a deployment-only repository for the Firelight Metanorma interactive viewer. It contains no application code—only `fl.zip` (pre-generated static site content) and a GitHub Actions workflow to deploy it to GitHub Pages.

## Repository Structure

- `fl.zip` - Pre-generated Firelight site; extracts to `fl/` directory containing static HTML/JS/CSS
- `.github/workflows/deploy.yml` - GitHub Actions workflow that deploys `fl/` to GitHub Pages

## Deployment Workflow

The `deploy.yml` workflow:
1. Triggers on push to `main`, pull requests, manual `workflow_dispatch`, and `repository_dispatch`
2. Extracts `fl.zip` into `fl/` directory
3. Uploads `fl/` as a GitHub Pages artifact
4. Deploys to GitHub Pages (only on `main` branch)

## Common Tasks

- **Update site content**: Replace `fl.zip` with a new generated version, then push to `main`
- **Trigger deployment**: Push to `main` or manually run the workflow via GitHub Actions UI
- **Preview locally**: Extract `fl.zip` and serve with any static file server

## Notes

- This repo has no build step, tests, or linting—it is solely a deployment conduit
- The site is served at `<org>.github.io/oiml-firelight/fl/` (content is in a `fl/` subdirectory)
- `fl.zip` is generated by the https://github.com/metanorma/firelight[Firelight] project
